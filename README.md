# Global Sales Store Analysis Dashboard

## Project Overview
This project is a Power BI dashboard developed to analyze global sales performance, customer behavior, and profitability trends.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key Features
- Sales Analysis
- Profit Analysis
- Regional Performance Tracking
- Customer Insights
- Interactive Filters and Slicers

## Business Insights
- Identified top-performing regions.
- Analyzed sales and profit trends.
- Compared category-wise performance.
- Evaluated customer purchasing behavior.
